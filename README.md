CNIT 325 Team Project
Darel Aradin, Ryan Hung, Linzzi Ji, Eunchae Jung, Priscilla Tam

Project Overview:
Our team will develop a Travel Planner application that helps users organize and plan trips, both domestically and internationally. Users will be able to select a city, browse attractions and points of interest, generate optimized travel routes using mapping services, and receive contextual recommendations like weather-based clothing tips. The system will integrate external APIs like Google Maps for route generation and location data, a weather service API for real-time forecasts, and a Hotel Search API for places to stay.
